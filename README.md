# FitnessForge

Firstly, ensure you have Node.js installed on your machine. Then, you can install the necessary dependencies by running the following command:
```bash
npm install
# ou avec yarn
yarn install
```
## Available scripts

### build
Build the application
```bash
npm run build
# ou avec yarn
yarn build
```

### dev front
Starts the development server for the front-end application
```bash
npm run dev:front
# ou avec yarn
yarn dev:front
```

### dev back
Starts the development server for the backend application
```bash
npm run dev:back
# ou avec yarn
yarn dev:back
```

### dev
Starts the development server for the front-end application
```bash
npm run dev
# ou avec yarn
yarn dev
```
